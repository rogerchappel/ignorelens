# Support

ignorelens is maintained as a public open source project by Roger Chappel.

## Questions and Bug Reports

Open a GitHub issue with:

- the package version or commit SHA
- your Node.js version and operating system
- the exact command you ran
- the expected result and actual result
- the smallest fixture or input that reproduces the problem

## Security Reports

Do not open public issues for vulnerabilities. Follow the process in [SECURITY.md](SECURITY.md).

## Maintenance Expectations

This project favors small, reviewable fixes with fixture-backed tests or smoke commands when behavior changes. Release-readiness-only updates should still include the verification commands that were run.
